# Nextjs Ecommerce App

> React, Next.js 를 활용한 간단한 쇼핑몰 앱입니다.
> Typescript, Redux, Prisma, Mysql

&nbsp;

## 사용 방법

```
# Clone Repository
$ git clone https://github.com/Kang95TH/new_ecommerce.git

# Move to Repository
$ cd new_ecommerce

# Install Dependencies
$ yarn

# Start App
$ yarn dev
```
